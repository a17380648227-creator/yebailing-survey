🚀 配置云端数据同步

让手机和电脑的数据实时保持一致

1 创建 GitHub Token

点击下方按钮打开 GitHub Token 创建页面（已自动填写好配置）

📝 在 GitHub 页面上的操作：
1. 确认描述为 "野百灵餐厅数据同步"
2. 确认已勾选 "gist" 权限
3. 点击页面底部的绿色按钮 "Generate token"
4. 立即复制生成的 token（只显示一次！）
2 粘贴 Token

将复制的 GitHub Token 粘贴到下方输入框：

3 保存配置

✅ 配置成功！

云端同步已启用！
现在所有设备（手机、电脑、平板）的数据将实时同步。

下一步：
1. 在手机上提交问卷 → 数据自动保存到云端
2. 在电脑上打开后台 → 自动显示手机提交的数据
3. 所有设备每30秒自动同步一次
